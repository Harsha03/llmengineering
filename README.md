# llmengineering

Minimal Python project for LLM engineering tasks.

## Structure
- `src/`: Source code and notebooks
- `tests/`: Test files

## Setup
- Uses [UV](https://github.com/astral-sh/uv) as the package manager
- Python >=3.11.7 required
- Jupyter notebook support enabled

## Environment
Set your environment variables in `.env` (e.g., `OPENAI_API_KEY=your-key-here`).
